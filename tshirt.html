T-Shirt and iPhone Project

Spenser Valentine

For this project I used the garment printer and the UV printer to make a T-Shirt and an iPhone case. For my T-shirt I created an image in Photoshop and exported it to the PDIP software which allowed me to print on a T-Shirt. For the UV printer I used the same image but exported it to a different software. These both worked well, with no bad results.
